1. 원본데이터(excel 파일 3개)  =>  data/original.py  =>  data/raw.csv

2. 데이터 전처리 raw.csv => notebooks/postprocessing.ipynb  => data/raw_postpro.csv

3. model/model2.py를 실행하면 model_label_encoders.pkl, model_softmax.pkl 생성
    model_label_encoders 는 LabelEncoder 객체가 담긴 딕셔너리 객체
    model_softmax는 인코딩 된 독립변수로 인코딩된 종속변수를 분류하는 모델 자체가 저장됨

4. DA 폴더에서 "flask run" 명령어 => http://localhost:5000/ 접속 후 form 에 값을 넣고 predict 버튼 클릭
