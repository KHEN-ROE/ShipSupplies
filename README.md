# ShipSupplies
> 선용품 최적 구매발주 데이터로 발주시간 예측

## 구조
 - DA, 데이터 분석 관련
 - FE, 사용자 입력 및 시각화 관련
 - BE, 인증 및 권한 그리고 API 관련
